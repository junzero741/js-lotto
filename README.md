## 🔥 Projects!

### STEP-1

<p>
<div>[x] 로또 구입 금액을 입력하면, 금액에 해당하는 로또를 발급해야 한다.</div>
<div>[x] 로또 1장의 가격은 1,000원이다.</div>
<div>[x] 소비자는 자동 구매를 할 수 있어야 한다.</div>
<div>[x] 복권 번호는 번호보기 토글 버튼을 클릭하면, 볼 수 있어야 한다.</div>
<div>[x] 위 기능들이 정상적으로 동작하는지 Cypress를 이용해 테스트한다.</div>
</p>
